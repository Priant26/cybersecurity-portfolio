# OSI Model (TryHackMe)

## Overview
This lab explained the OSI Model and how network communication is divided into layers.

## What I learned
- The purpose of the OSI Model
- The seven layers of the OSI Model
- How each layer has a specific role in communication
- How the OSI Model helps with troubleshooting

## What I practiced
- Understanding which network tasks belong to each layer
- Mapping real-world examples to OSI layers

## Why this matters
The OSI Model helps cybersecurity professionals understand where network issues or attacks occur and makes troubleshooting more effective.
