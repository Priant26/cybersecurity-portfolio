# Linux Fundamentals Part 1 (TryHackMe)

## Overview
This lab introduced the basics of the Linux operating system and how to navigate and interact with it using the command line.

## What I learned
- What Linux is and why it is widely used
- How to navigate directories using basic commands
- How to list files and folders
- How to view the current working directory
- Basic understanding of the Linux file system structure

## Commands practiced
- pwd
- ls
- cd
- clear

## What I practiced
- Navigating through the Linux file system
- Using the terminal instead of a graphical interface
- Understanding how Linux organizes files and directories

## Why this matters
Linux is widely used in servers, networking, and cybersecurity.  
Understanding basic Linux commands is essential for working with security tools, servers, and system logs.
