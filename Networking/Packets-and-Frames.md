# Packets and Frames (TryHackMe)

## Overview
This lab explained how data is broken into smaller units when traveling across a network.

## What I learned
- What packets and frames are
- The difference between packets and frames
- How data is prepared for transmission
- Why breaking data into smaller pieces is important

## What I practiced
- Understanding how data moves through a network
- Learning how packets and frames work at different layers

## Why this matters
Packets and frames are core concepts in networking and cybersecurity because all network traffic is built using them.
